## fastify-log-filters monorepo

This repository contains the `fastify-log-filters` Fastify plugin and related demo applications.

- `packages/fastify-log-filters`: Core plugin package
- `demos/`: Example Fastify apps showcasing the plugin (to be added)

### Development

```bash
pnpm install
pnpm dev
pnpm tsc # typescript-native
pnpm build
pnpm lint # oxlint
pnpm lint:fix
pnpm format # oxfmt
pnpm test # vitest
```
